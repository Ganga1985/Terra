# Terraform Remote State Management

## Project Goal

Configure remote Terraform state using Amazon S3 and DynamoDB.

## Features

* S3 backend
* DynamoDB locking
* Remote state storage
* Team-ready Terraform workflow

## Skills Demonstrated

* S3
* DynamoDB
* State Management
* Infrastructure as Code
