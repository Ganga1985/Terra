# Modules Lab

## Objective

Create and use a reusable Terraform module to deploy an EC2 instance.

## Steps Performed

1. Created an EC2 module.
2. Defined module input variables.
3. Defined module outputs.
4. Called the module from the root configuration.
5. Deployed an EC2 instance using the module.
6. Verified deployment in AWS.
7. Destroyed resources after testing.

## Outcome

Successfully deployed an EC2 instance using a reusable Terraform module.

## Skills Practiced

* Terraform Modules
* Module Variables
* Module Outputs
* AWS EC2 Deployment
* Terraform Code Reusability
