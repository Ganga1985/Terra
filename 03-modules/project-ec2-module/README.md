# EC2 Deployment Using Terraform Modules

## Project Goal

Deploy an Amazon EC2 instance using reusable Terraform modules.

## Features

* Reusable EC2 module
* Module input variables
* Module outputs
* Simplified infrastructure management

## Files

* main.tf
* variables.tf
* outputs.tf
* modules/

## Skills Demonstrated

* Terraform Modules
* Module Inputs and Outputs
* AWS EC2 Provisioning
* Infrastructure as Code
