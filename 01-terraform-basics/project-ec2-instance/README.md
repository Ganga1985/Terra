# AWS EC2 Instance Deployment with Terraform

## Project Overview

This project deploys an Amazon EC2 instance on AWS using Terraform.

## Files

* `main.tf` - Terraform configuration for the EC2 instance

## Terraform Workflow

```bash
terraform init
terraform plan
terraform apply
terraform destroy
```

## Outcome

Successfully provisioned an EC2 instance using Infrastructure as Code (IaC).

## Skills Demonstrated

* Terraform Configuration
* AWS Provider Setup
* EC2 Provisioning
* Infrastructure as Code
* Terraform State Management

## Verification

EC2 instance successfully deployed using Terraform.
