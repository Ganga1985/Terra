# EC2 Deployment Lab

## Objective

Deploy an Amazon EC2 instance using Terraform.

## Steps

1. Configured the AWS provider.
2. Created an EC2 resource.
3. Ran `terraform init`.
4. Ran `terraform plan`.
5. Ran `terraform apply`.
6. Verified the instance in AWS.

## Outcome

Successfully deployed an EC2 instance using Terraform.

## Skills Practiced

* AWS Provider Configuration
* EC2 Provisioning
* Terraform Workflow
* Infrastructure as Code
