# Provisioners Lab

## Objective

Use a Terraform provisioner to execute a command during resource creation.

## Steps Performed

1. Created an EC2 instance.
2. Configured a local-exec provisioner.
3. Executed a command during deployment.
4. Verified Terraform output.
5. Destroyed resources after testing.

## Outcome

Successfully automated a post-deployment task using a Terraform provisioner.

## Skills Practiced

* Terraform Provisioners
* local-exec
* AWS EC2 Deployment
* Infrastructure Automation
