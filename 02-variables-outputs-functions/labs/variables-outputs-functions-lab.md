# Variables, Outputs and Functions Lab

## Objective

Deploy an EC2 instance using Terraform variables, outputs, and functions.

## Steps Performed

1. Created input variables for AWS region and instance type.
2. Used variables in the Terraform configuration.
3. Created outputs to display instance information.
4. Used the `upper()` function for resource tagging.
5. Deployed the EC2 instance.
6. Verified deployment in AWS.
7. Destroyed resources after testing.

## Outcome

Successfully deployed an EC2 instance using reusable Terraform variables and outputs.

## Skills Practiced

* Terraform Variables
* Terraform Outputs
* Terraform Functions
* AWS EC2 Deployment
* Infrastructure as Code
