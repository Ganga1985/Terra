# EC2 Deployment with Variables and Outputs

## Project Goal

Deploy an Amazon EC2 instance using Terraform variables, outputs, and functions.

## Features

- AWS region configured using variables
- EC2 instance type configured using variables
- Outputs display deployment information
- Terraform function used for resource tagging

## Files

- main.tf
- variables.tf
- outputs.tf

## Skills Demonstrated

- Terraform Variables
- Terraform Outputs
- Terraform Functions
- AWS EC2 Provisioning
- Infrastructure as Code
